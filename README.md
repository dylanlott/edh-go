# EDH-Go
> A Magic: The Gathering board state tracker built with GraphQL and Go.
 [![Build Status](https://travis-ci.org/dylanlott/edh-go.svg?branch=master)](https://travis-ci.org/dylanlott/edh-go)

## Running

Run server:

```
$ docker-compose up -d --build
```

Run Vue app:

```
$ cd frontend
$ npm run start
```
